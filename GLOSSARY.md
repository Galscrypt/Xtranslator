# Custom Glossary

Актуальный пользовательский словарь для `X Translator`.

Как использовать:
- Открой popup расширения.
- Нажми `Load glossary`.
- При необходимости обнови JSON ниже.
- Нажми `Apply glossary`.

Ниже production-версия словаря: очищена от конфликтов между секциями и агрессивных замен.
